# pyCalc

Uma calculadora simples em Python, com modo interativo:
- **Modo Interativo**: Interface de linha de comando amigável, com entrada de dados via `input()`.

## Como usar

Execute o script sem argumentos:

```bash
py pyCalc.py
```


Siga as instruções na tela. Você pode digitar `sair` tanto no campo do nome quanto na operação para encerrar.

## Requisitos
- Python 3.x

## Autor
- Projeto de exemplo para estudos de Python CLI e interatividade.
